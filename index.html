About

Qalaxy Quest; is a game based on the popular Space Invaders/Galaxian games. The game is written in Javascript as an attempt to further familiarise myself with the language in conjunction with HTML5 and CSS3. The game consists of a welcome screen that shows the level, whether the music is muted or not, score as well as demonstrating the controls and lives the player has. During game play once the player has succesfully defeated a wave there will be an announcement to notify the player they have succesfully passed the level and will proceed onto the next. With each increase the enemies rate of fire and speed will increase as well as the speed of the background panning to give the illusion of the player travelling faster.

How does it work?

Qalaxy Quest; The game consists of mulitple canvas elements that are on top of one another to reduce running cost and to significantly improve the games performance. An object was created in Javscript to ensure that all the images needed for the game have successfully loaded which will then start the game. To produce the panning background I set the image to scroll at a given speed following which it will reset itself once the image as scrolled off screen. The next step was to create the player object which was drawn on the "ship" canvas where I determined the players; speed, rate of fire, as well as the ships action via assigning movement to key codes as well as making sure the player stays within the screen. The third step was creating the enemy object which would inimate the movements of the space invaders alien except for moving downward like the orginal game. Again as with the player ship I ensured that the aliens stayed within the screen bounderies with a percentage of fire variable to make it so theres no fixed pattern for the player to memorise. The next stage was to implement spatial partitioning for the collision detection aspect of the game, which I looked at on Gamedevtus+. The invovles taking a confined space and subdividing it into mutliple smaller spaces, this allows for quicker collision detections. Assigning a var iscolliding I am able to specific objects I was able to determine what can collide with what for instance player being hit by the enemeies bullets and vice versa. The last step was adding wav and mp3 files for the bullet sound effects as well as for the game during play, succesfully completeling a level and for when the player dies.
